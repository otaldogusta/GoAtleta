## Escala de Score

- 1 = Ruim
- 2 = Fraco
- 3 = Aceitavel
- 4 = Bom
- 5 = Muito bom

Meta por turma: media final >= 4.0

## Rubrica (aplicar em cada aula)

| Criterio | Pergunta de validacao | Score (1-5) | Evidencia curta |
| --- | --- | --- | --- |
| Forma de jogo | Parece Mini 2x2, 3x3 ou 4x4 da categoria correta? |  |  |
| Linguagem | Fala a lingua da quadra e da faixa etaria? |  |  |
| Progressao | Parece continuidade real do mes/fase? |  |  |
| Aplicabilidade | Um professor aplicaria sem reescrever tudo? |  |  |
| Idade | Adequado para a faixa sem ficar adulto ou infantil demais? |  |  |

## Checklist de Reprovacao Imediata

Reprovar a aula na hora se ocorrer qualquer item abaixo:

- 08-10 com linguagem tatica abstrata
- Mini 2x2 com cara de 6x6
- Mini 3x3 pobre demais como iniciacao muito infantil
- Mini 4x4 sem cara de transicao
- Plano bem escrito, mas sem identidade da categoria
